## 快速使用

MolGen 使用说明
1. 输入分子
填入目标分子 SMILES，软件会自动锁定母核骨架。
2. 设置生成数量
默认生成 30 个分子，可自定义 10~500 个。
3. 性质不突变约束
可调整：
分子量波动范围
LogP 波动范围
TPSA 极性波动
氢键受体 / 供体波动
确保生成分子与原化合物性质接近，不突变。
4. 生成结果
2D 结构预览
成药性评分（0~5 分）
合成难度等级
经济型逆合成路线
导出 CSV / SDF 分子库
5. 应用场景
药物分子设计
分子对接及虚拟筛选化合物库构建
候选药优化
药物分子发现
专利规避分子设计
构效关系研究
